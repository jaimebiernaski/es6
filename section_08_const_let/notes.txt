**** LET & VAR ****

The turns the js code more clear, more legible and easy to undestand
In es6, we does not use 'var' anymore.

Istead use:

const: variables that will never changes
let: variables that change over time

// ES5
var name = 'Jane';
var title = 'Software Engineer';
var hourlyWage = 40;

// ES6
const name = 'Jane';
let title = 'Software Engineer';
let hourlyWage = 40;
